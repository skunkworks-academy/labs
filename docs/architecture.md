# Skunkworks Academy Labs Architecture

## Purpose
This repository is the public front door for Labs on Demand.

## Linked platform components
- labs-catalog: canonical lab catalogue and metadata
- labs-guides: learner-facing lab guides
- labs-provisioning: infrastructure-as-code and environment lifecycle
- labs-access-gateway: browser-based access layer
- labs-booking: cohort and learner booking workflows
- labs-evidence: assessment evidence, rubrics and badge mappings
- academy-portal: learner, instructor and admin dashboard
- academy-validation: cross-site QA and deployment validation
